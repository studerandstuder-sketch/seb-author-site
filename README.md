# Sébastien Studer — Author Website

Horowitz-style author site built with Next.js (App Router) + Tailwind.
Deployed on Netlify or Vercel.

## Quickstart

```bash
pnpm i   # or npm install / yarn
pnpm dev # or npm run dev
```

## Build

```bash
pnpm run build
pnpm start
```

## Deploy to Netlify
- Connect the repo
- Build command: `npm run build`
- Publish directory: `.next`
- Node version: 18+
